# Plant-Simulator-
Code that lets the user go through a list of plant types. Eventually, depending on the inputs the code tells them to use, the program will output how to care for the selected plant.

The code utilizes:

    1. 2D Array to hold the type fo plant, indoor or outdoor, 
    which is why the inputs are numbers for easy selection

    2. A method and an input exception method to organize code 
    and help reduce redundancy

    3. Loops to help the user back track on some steps and
    help with incorrect input exception
